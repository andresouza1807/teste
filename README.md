# Plataforma IA + RPA Docs (Template)

Este repositório é um template profissional no estilo **Docs-as-Code**, pronto para uso com:

- MkDocs + Material
- Mermaid (diagramas)
- C4 Model
- Runbooks operacionais
- ADR (Architecture Decision Records)

## Rodar localmente

```bash
pip install mkdocs-material
mkdocs serve
```

Acesse:
http://127.0.0.1:8000

## Publicar no GitHub Pages

```bash
mkdocs gh-deploy
```
