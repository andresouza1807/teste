# Threat Model (STRIDE)

## Ameaças comuns
- Spoofing (falsificação de identidade)
- Tampering (alteração de dados)
- Repudiation (negação de ações)
- Information Disclosure (vazamento)
- Denial of Service (DoS)
- Elevation of Privilege

## Mitigações
- autenticação forte
- criptografia
- auditoria
- rate limits
